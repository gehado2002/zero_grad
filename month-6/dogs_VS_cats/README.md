# 🐶🐱 Dogs vs Cats Image Classification

A deep learning project for **binary image classification** (Dogs vs Cats) using **CNN** and **VGG16 Transfer Learning**, implemented on **Google Colab**.

---

## 📌 Overview
- **Dataset:** 25,000 images (balanced)
- **Classes:** Dog 🐶 | Cat 🐱
- **Framework:** TensorFlow / Keras
- **Platform:** Google Colab

---

## 📁 Data Split
- **Training:** 20,000 images (80%)
- **Validation:** 5,000 images (20%)

---

## 🧠 Models
- **Custom CNN:** ~86% validation accuracy  
- **VGG16 (Transfer Learning + Fine-Tuning):**
  - **Train Accuracy:** 97.22%
  - **Validation Accuracy:** 96.42%

---

## 📊 Final Results
- Accuracy: 96%
  - Cat → Precision: 0.96 | Recall: 0.97
  - Dog → Precision: 0.97 | Recall: 0.96
